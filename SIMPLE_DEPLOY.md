# 最简单部署方式

如果你觉得部署太复杂，就按这个版本来。

## 你只需要准备

1. 一台云服务器，系统选 Ubuntu。
2. 如果暂时没有域名，也可以先用服务器公网 IP 访问。

## 给服务器开放端口

在云服务器控制台的安全组里开放：

- `80`
- `443`
- 如果暂时没有域名，还要开放 `8000`

## 上传项目

把整个项目文件夹上传到服务器。

## 运行一条命令

进入项目目录后执行：

```bash
bash deploy/start_server.sh
```

脚本会问你：

- 域名或服务器公网 IP
- 负责人 `admin` 的初始密码

然后它会自动安装 Docker 并启动系统。

## 登录地址

如果你有域名：

```text
https://你的域名/app/
```

如果你暂时没有域名：

```text
http://服务器公网IP:8000/app/
```

账号：

```text
admin
```

密码就是你运行脚本时输入的负责人密码。

## 更简单的现实做法

你可以把这个文件发给学校网管、懂电脑的学生或云服务器客服，让他们按这里操作。
